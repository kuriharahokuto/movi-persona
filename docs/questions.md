# MOVI Persona

Question Specification

---

# Overview

Purpose

MOVI Persona Archetype を判定するための質問セット。

---

Structure

Total Questions

16

---

Question Format

- ACT番号
- シーンタイトル
- 映画ジャンル
- シーン描写
- 選択肢

---

Scene Rules

- シーン描写は3〜6行
- 16問すべて異なる映画ジャンル
- 選択肢は主人公の行動として記述する
- 選択肢に唯一の正解を作らない
- 選択肢は分析文ではなく、主人公の決意や覚悟として記述する
- スコアリング配点は scoring_engine.md に従う

建造物が遠ざかっても、帰れる場所を失うわけにはいかない。
