# Result Hero Poster Image Rule

The result screen must display the archetype image.

Image source:

archetype.imagePath

Display location:

Top of Result Reveal Screen

and

Hero Poster section of Full Report Screen

Rules:

- Use the image matching the final archetype
- Do not generate images dynamically
- Do not call external image APIs
- Use local image assets only
- Always provide alt text
