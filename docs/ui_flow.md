# MOVI Persona

User Experience Flow Specification

---

# Purpose

本ドキュメントは

MOVI Persona

のユーザー体験を定義する。

目的は

- 離脱率の低下
    
- 診断完了率の向上
    
- 結果共有率の向上
    
- 再訪率の向上
    

である。

---

# Design Philosophy

ユーザーは

性格診断を受ける

のではない。

---

ユーザーは

映画の主人公になる。

---

サービス全体を

映画予告編

として設計する。

# Anti Drop Rules

Never show:

- Login Wall
    
- Signup Requirement
    
- Payment Screen
    
- Ads Before Result
    

---

Result must be reachable

within 3 minutes.

---

# Success Metrics

---

Average Session Time

4 Minutes+

# Final Rule

Every screen must answer:

Why should the user continue?

If a screen does not increase curiosity,

it should be removed.
