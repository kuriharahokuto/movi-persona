# Result Templates v2

Version: 2.0

## Purpose

診断結果を

「映画ポスター」

として体験させる。

結果ページは単なる診断結果ではなく、

ユーザー自身を主人公とした映画の設定資料集として設計する。

# Model Consistency Rule

Version 3.0

The following legacy axes are deprecated:

- Exploration
    
- AuthorityResistance
    
- StrategicThinking
    
- Conviction
    
- Acceptance
    
- Order
    
- Independence
    
- Emotionality
    

These terms must not be used for:

- Score calculation
    
- Result rendering
    
- User-facing reports
    

Current model:

Core Dimensions

- EXP_PRAG
    
- EMP_STR
    
- REB_GUA
    
- IDE_REA
    

Secondary Traits

- RISK
    
- SACRIFICE
    
- COLLECTIVISM
    
- JUSTICE
    

All reports must use the current model only.

# Character Overview

Length

300〜500 words

Purpose

ユーザーが

「自分に当てはまる」

と感じること。

Structure

- 基本的な価値観
    
- 判断基準
    
- 対人関係
    
- 成功パターン
    
- ストレス時の行動
    

Example

あなたは現状維持よりも変化を好みます。

周囲が諦める状況でも、  
可能性が残っている限り挑戦を続ける傾向があります。

他人からは大胆だと思われますが、  
本人は合理的な判断をしているつもりであることが多いでしょう。

また、ルールそのものより、  
そのルールが何のために存在するかを重視します。

そのため理不尽な権威には反発しますが、  
尊敬できる人物には強い忠誠心を示します。

# Screenwriter Report

Title

Screenwriter Analysis

Purpose

映画脚本家視点から解説する。

Example

もしあなたが映画の主人公なら、

第1幕では平凡な日常に違和感を抱き、

第2幕で未知の世界へ飛び込み、

第3幕では仲間を率いて大きな選択を行うでしょう。

物語の終盤では、

現状維持と変革の間で葛藤する場面が描かれる可能性があります。

あなたは典型的な変革者型主人公です。

# Behavioral Intelligence Report

Title

Behavioral Intelligence Report

Purpose

情報機関の人物分析風レポート

Important

実在のCIAやFBIを名乗らない。

架空レポート形式とする。

Example

Subject Analysis

対象者は未知環境においても  
探索行動を選択する傾向が高い。

リスク認識能力は保持しているが、  
好奇心が意思決定へ強い影響を与える。

集団内では先導者として振る舞うことが多い。

危機的状況では行動停止よりも問題解決を優先する。

Assessment

High Initiative

High Independence

Moderate Risk Exposure
