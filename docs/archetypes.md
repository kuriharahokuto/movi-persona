Strengths:

- 創造力
    
- 挑戦心
    
- 適応力
    
- 希望を失わない
    

Weaknesses:

- 飽きやすい
    
- 現実逃避しやすい
    
- 計画不足
    
- 衝動的
    
- 危険を軽く見やすい

ルーティンワークは苦手。

Relationship Style:

相手と一緒に成長したいと考える。

Behavior Pattern:

未知の選択肢を優先する傾向がある。

Movie Examples:

- Avatar

# Archetype Rules

Each archetype must provide enough source data to render:

- Hero Poster
    
- Character Overview
    
- Strengths
    
- Weaknesses
    
- Behavioral Analysis
    
- Screenwriter Report
    
- Behavioral Intelligence Report
    
- Movie Recommendations
    
MVP Required Fields

For each archetype, the following fields are mandatory:

- Japanese Name
    
- Tagline
    
- Core Motivation
    
- Core Fear
    
- Narrative Role
    
- Movie Examples
    
The following fields are recommended, but may be derived from result_templates.md during MVP:

- Strengths
    
- Weaknesses
    
- Work Style
    
- Relationship Style
    
- Behavior Pattern
    

The actual user-facing text is generated from result_templates.md using this archetype definition.

Image Priority:
Hero Poster

Usage:
- Result Reveal Screen
- Full Report Screen
- Share Card
